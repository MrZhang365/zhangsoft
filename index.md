# 小张软件
## 关于
小张软件是由小张于2019年创立的软件团队，是他的梦想。在发展过程中遇到了不少挫折（点名kiddo），经历过一次大规模删库~~，但是都坚持了下去。常用编程语言：Visual Basic 6.0和Python3。

我们相信小张软件会变得越来越好！
